# DataSoluTech: Conteneurisation via Docker d'une BDD MongoDB et son script import Python

Ce projet a pour objectif d'automatiser la **migration d’un dataset CSV vers MongoDB**, tout en garantissant la portabilité et la scalabilité grâce à **Docker**.  
L’environnement inclut également une phase de **tests automatiques avec Pytest** pour valider l’import.
